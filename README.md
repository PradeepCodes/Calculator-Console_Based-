# Java Console Calculator

## Description

This is a simple Java console-based calculator application that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.
The program takes user input from the console and performs the selected operation using a switch statement.

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Division-by-zero handling
* Simple and beginner-friendly Java implementation

## Technologies Used

* Java
* Scanner class for input handling
* Switch-case control structure

## How to Run

1. Clone the repository:

   ```
   git clone https://github.com/PradeepCodes/Calculator-Console_Based-
   ```
2. Open the project in any Java IDE (IntelliJ, Eclipse, VS Code).
3. Compile and run the `Calculator.java` file.
4. Enter the numbers and choose the operator when prompted.

## Sample Input / Output

```
Enter the First Number
10
Enter the Second Number
5
Choose the operator(+,-,*,/)

```
